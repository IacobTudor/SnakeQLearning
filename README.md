# Snake Q-Learning (C++ / SFML)

Reinforcement learning Snake AI using a Q-learning table and SFML visualization.

The agent learns by:
- getting reward for eating food
- getting penalty for collisions
- small reward for moving closer to food
- updating Q-values over time

## Features
- Real-time SFML rendering
- Grid-based environment
- Q-learning from scratch
- Training + visual mode

## Tech
C++20, SFML, CMake, vcpkg

## Build
cmake -B build  
cmake --build build

Learning project for reinforcement learning.
